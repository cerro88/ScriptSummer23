Etiqueta span

La etiqueta span sirve para cambiar el estilo a una parte del texto

Dentro de la etiqueta span de apertura introduciremos el estilo que queremos dar a esa parte del texto

También podemos usar un selector de etiquetas para diseñar todas las etiquetas span y si hay demasiadas podemos usar selectores de ID, por ejemplo span class='textodestacado', luego podemos usar selectores de clase: .textodestacado y darle estilos css en nuestro archivo css

La etiqueta span tiene etiqueta de apertura y de cierre.
