PÁRRAFOS

Un párrafo en HTML se crea utilizando la etiqueta"p". Esta etiqueta se utiliza para definir un bloque de texto que representa un párrafo en una página web.

Podemos escribir tantos parrafos como queramos.

IMAGENES Y ENLACES

Para insertar una imagen etiqueta 'img + atributo src, donde se coloca el enlace limpio de la imagen, atributo alt para añadir una descripción de la imagen y los atributos width y height para definir el alto y ancho de la imagen


Para insertar un enlace en la página usamos la etiqueta 'a' + el atributo href donde colocamos la ulr de la página, se cierra la etiqueta; y entre la etiqueta de apertura y la de cierre pondremos el texto que queremos que aparezca en el enlace
