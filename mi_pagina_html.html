INICIO A HTML CON MIMO

Con lo aprendido en clase y en la aplicación el objetivo es practicar las diferentes etiquetas y estilos de CSS

La etiqueta div divide en secciones, es decir lo que va dentro de div esta separado del resto del código, si en el archivo css de styles aplico una regla a div se aplicará para todo lo que engloba div

PÁRRAFOS

Un párrafo en HTML se crea utilizando la etiqueta"p". Esta etiqueta se utiliza para definir un bloque de texto que representa un párrafo en una página web.

Podemos escribir tantos parrafos como queramos.

IMAGENES Y ENLACES

Para insertar una imagen etiqueta 'img + atributo src, donde se coloca el enlace limpio de la imagen, atributo alt para añadir una descripción de la imagen y los atributos width y height para definir el alto y ancho de la imagen


Para insertar un enlace en la página usamos la etiqueta 'a' + el atributo href donde colocamos la ulr de la página, se cierra la etiqueta; y entre la etiqueta de apertura y la de cierre pondremos el texto que queremos que aparezca en el enlace

CSS BÁSICO

Le podemos dar estilo a nuestras etiquetas añadiendo el atributo 'style' dado que style es un atributo, necesitaremos un = para darle un valor

Los valores de los atributos siempre van entre comillas

Se hará en la etiqueta de apertura: por ejemplo 'h3 style="color:red;"'titulo'/h3' Cuando añadimos color:pink como valor, estamos haciendo una declaración. Una declaración esta formada por una propiedad como color y un valor como es pink, esto va entre comillas y separador por: y finaliza con ;
